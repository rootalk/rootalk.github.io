## Install Theme

> Theme folder is not in this repo by default. Run this command at the first time.

```sh
# 更新主题
# rm -rf themes/minos

git clone https://github.com/glenhooo/hexo-theme-minos.git themes/minos
cp themes/minos/_config.example.yml themes/minos/_config.yml
# node init_script.js
```

## Development

```sh
npm run serve # hexo s
```

## Generate Static

```sh
npm run build # hexo g
```
